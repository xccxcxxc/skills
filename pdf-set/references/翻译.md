# 翻译

当前默认 prompt 为**法语→中文对照翻译**；若翻译其他语言，必须提供匹配的 `--prompt-file`，不要把默认 prompt 当通用翻译器。

## 配置（推荐环境变量）

```text
PDF_TRANSLATE_BASE_URL=...
PDF_TRANSLATE_API_KEY=...
PDF_TRANSLATE_MODEL=...
```

本地 ignored 的 `assets/secrets_openai.txt` 仍兼容，但不得提交真实密钥。脚本只在真正发请求时校验配置，因此 `--help` 无密钥也可使用。

## 输入/输出
- 输入：`translate-typeset/N.md`
- 输出：`translate-result/N.md`
- 默认每次 5 段，可用 `--chunk-size`。

```bash
python scripts/translation.py --base-dir "书籍目录"
```

单文件模式必须同时指定 `--input-file` 和 `--output-file`。

## 完整性
- 输出按原文/引用译文/`---` 结构校验；段数不匹配时不得冒充完成。
- API `finish_reason=length/max_tokens/content_filter` 视为失败并重试/退出，不接受截断译文。
- 断点续翻按已完成分隔块计算；不要手工截断输出文件。
- Markdown 图片和 HTML 表在原文侧只出现一次，译文侧按 prompt 使用空引用占位。

## 参数
`--base-dir`、`--book-name`、`--input-dir/file`、`--output-dir/file`、`--start/end`、`--chunk-size`、`--prompt-file` 及对应 `*-from`。
