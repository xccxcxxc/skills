# 安装前置组件

Python 依赖：

```bash
python -m pip install -r pdf-set/requirements.txt
```

Alpine/minis 若 PyPI wheel 不可用，优先安装系统包，再补纯 Python 包：

```bash
apk search py3-pillow py3-pypdfium2 py3-openai
```

必需：
- Python 3.10+
- `openai`
- `pypdfium2`
- `Pillow`
- `pandoc` 3.x（导出 EPUB；不属于 Python 包）

可选：
- Java + 官方 EPUBCheck（严格兼容性复核）；没有时使用 `scripts/validate_epub.py` 作为最低门槛。

检查：

```bash
python -c "import openai, pypdfium2, PIL; print('python deps ok')"
pandoc --version
```

不要把 API key 写入仓库。
