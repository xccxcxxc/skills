# 排版成书

输入：已完成标题分类的 `merge-result/0.rough.md`。输出：`<书名>.md`。

```bash
python scripts/typeset_book.py --base-dir "书籍目录"
```

## 忠实规则
- 保留第一个 H1 之前的 YAML metadata、作者、前言等内容；不再静默丢弃 preamble。
- 默认不删除重复字词，不自动把 `and/or/he/of` 替换成中文。
- 旧 token 修复仅在明确检查过 diff 时显式使用：

```bash
--repair-inline-tokens
```

- GFM 表按块保留；HTML 表仅识别 `<div class="table-wrap">` / `<table>`，不把普通 `<div>` 当表格。
- HTML `rowspan/colspan` 保留。
- `--keep-intermediate` 可保留分章与排版中间文件；用户验收 EPUB 前仍应保留整个工作目录。

## 插图占位处理
导出前必须逐项处理 `🀄️页码.jpg`：
- 真实正文配图/地图/图版：复制到 `assets/`，替换成真实图片引用；
- 纯装饰、空白、重复封面、目录扫描页：删除占位；
- 封面只通过 `assets/cover.jpg` + `--epub-cover-image` 提供一次，不在正文重复嵌入。

## 检查
- 所有应导航标题是 H1；
- `🀄️/🈳` 清零后才允许严格 EPUB 验收；
- 没有重复封面；
- GFM/HTML 表结构校验通过；
- YAML/前言未丢失；
- 不存在静默删字/改词。
