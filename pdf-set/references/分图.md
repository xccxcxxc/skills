# 分图

## 默认命令

```bash
python scripts/convert_pdf_to_images.py "书名.pdf" "书籍目录/images" \
  --dpi 144 --format jpg --jpeg-quality 92 --start-index 0
```

默认特性：
- 144 DPI；
- JPEG quality 92、4:4:4（`subsampling=0`），适合小字和表线；
- 文件名从 `0.jpg` 开始；
- 生成 `images/manifest.json`，记录 PDF hash、页数、DPI、图片 hash；
- 全本转换时若发现超出当前 PDF 页数的旧数字图片，报错而不是混入旧页。

## 页范围
`--start-index` 只控制输出文件编号；真正选择 PDF 页使用：

```bash
--first-page 10 --last-page 20
```

旧 `--start` 仅为 `--start-index` 的兼容别名。

重建已有图片使用 `--overwrite`。缩小大图可用 `--max-dim`，采用 LANCZOS；OCR 通常不建议过度缩小。
