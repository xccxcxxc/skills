# OCR 指令

## 1. 忠实边界（最高优先级）
- 识别到什么就输出什么：不改写、不补全、不润色、不猜测。
- 保留本页正文第一个和最后一个可见字符；页末残句、连字符、逗号等原样保留。
- 简体保持简体，繁体保持繁体；不做语言替换。
- 删除页眉、页脚、页码、边码和每页重复的章节标题，但不要删除正文中的真实标题。
- 空白页输出 `🈳+文件名`。

## 2. 阅读顺序与段落
- 按原页阅读顺序输出：正文、表题、表格、资料来源/表注、后续正文。
- 正文段落之间用一个空行（`\n\n`）；不要用单换行模拟段落。
- 新段首加两个半角空格；上一页未完段的续接不要加缩进。
- 章节/小标题用 `##`；不要使用 `#` 或 `###`（目录页的标题不加 `##`）。
- 图片、照片、地图、纯图示、无法可靠文本化的复杂公式/表格：在原位置输出 `🀄️+文件名`；不要用 Markdown 图片语法。

## 3. 表格选择
只要行列可辨，就文本化；不要因为页面同时有表格和正文而整页输出 `🀄️`。

### A. 单层表头或长文本叙述表 → GFM 管道表
适用：姓名、职务、学校、经历、轨迹等长文本列。
- 表头一行、分隔行一行、数据行若干；每行以 `|` 开始和结束。
- 列数必须一致。
- 单元格源文本内不插入硬换行；阅读器可自动折行。
- 不加 `table-dense`。

示例：

```markdown
| 姓名 | 职务 | 任前职务 | 轨迹 | 时间差② |
| --- | --- | --- | --- | :---: |
| 郭易声 | 李庄镇镇长 | 大门镇党委副书记 | 团县委副书记—大门镇党委副书记—李庄镇镇长 | 10 |
```

### B. 多级表头、跨列合并、统计密表 → 白名单 HTML
- 外层必须是 `<div class="table-wrap">`。
- `<table class="table-dense">`；使用 `<thead>`、`<tbody>`、`<tr>`、`<th>`、`<td>`。
- 用数字 `rowspan`/`colspan` 还原原表合并结构；禁止把多级表头压扁后丢失上层分组。
- 每个 `th/td` 内仅一行纯文本；空格写空标签；禁止 `<br>`、`<p>`、`style`、事件属性或其他 HTML。
- 每个开始标签都必须有对应结束标签；每行有效列宽（含 rowspan/colspan）一致。

示例：

```html
<div class="table-wrap">
<table class="table-dense">
<thead>
<tr><th rowspan="2"></th><th colspan="2">时间差</th><th colspan="2">轨迹类型</th></tr>
<tr><th>0-5</th><th>6-10</th><th>单一</th><th>混合</th></tr>
</thead>
<tbody>
<tr><td>乡镇长（15人）</td><td>4</td><td>7</td><td>2</td><td>13</td></tr>
</tbody>
</table>
</div>
```

### C. 单层但列很多
- 优先 GFM；若原表上层分组对理解不可缺少，则按 B 输出 HTML。

## 4. 表内文字与注释
- 单元格只写原格可见字/数；不扩写，不臆造 `-`、百分号或注释。
- 数字、区间、标点按原表；原表无 `%` 不补 `%`。
- 表内注号仅保留符号（①②* 等）；脚注内容移到表外 `<sup>【…】</sup>`。
- 表题放在表格上方独立段落，不放进表头。
- 表格内部不插空行；表前表后各一个空行。

## 5. 跨页表格
- 起表页：输出完整表头和本页可见数据。
- 续表页：重复与上页完全相同的表头结构，再输出本页新数据；不要重复上页数据。
- GFM 续表重复 header/separator；HTML 续表重复 `<thead>`（rowspan/colspan 一致）。
- 本页有几行就输出几行，不补全不可见数据；资料来源只在原页出现的位置输出。
- 后处理会在表头结构完全一致时合并相邻续表。

## 6. 注释
### 边注
- 与正文同处但字号明显更小的括号说明，格式化为 `（注：原文）` 放在原位置。
- 跨页未完只保留本页可见括号，不补另一半。

### 尾注
- 文内注号改为 `<sup>【原注号】</sup>`；章节末/书末注释正文保持原样，不内嵌、不添加模型推断的标题。

### 脚注
- 本页页脚有与正文注号对应的注释时：删除原注号和页脚副本，把注释移到正文对应位置，格式为 `<sup>【原文】</sup>`。
- 注释内容不分段；本页明显未完时末尾加 `⬇️`。
- 本页只有上页脚注续文时，放页首：

```markdown
  <sup>⬆️原文</sup>
```

## 7. 输出检查（模型自检）
- 没有代码围栏包裹最终输出。
- 没有添加解释、总结或“识别结果如下”。
- GFM 列数一致；HTML 标签配对、rowspan/colspan 有效。
- 表格外正文首尾字符没有因表格存在而丢失。
